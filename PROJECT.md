# Skeramos — Творческий дом

## What This Is

Интерактивный сайт-визитка для Skeramos — творческого пространства в Бишкеке, объединяющего гончарную студию с мастер-классами и уютный мини-отель с романтическими сценариями. Сайт с премиальным дизайном в стиле Этно-Нео, максимально живыми анимациями и уникальным переключателем между двумя мирами (Творчество / Отель).

## Core Value

**Погружение в атмосферу с первого клика** — сайт должен передавать эмоции и уникальность Skeramos через визуальный сторителлинг и живые анимации, превращая посетителей в клиентов ещё до первого звонка.

## Context

**О компании:**
- Skeramos — творческий дом в Бишкеке (ул. Шукурова 8)
- Слоган: "Лепим счастье"
- Два направления: гончарная студия + мини-отель
- Этно-атмосфера: тёплый свет, натуральные фактуры, внимание к деталям
- Активный Instagram: @skeramos.kg

**Конкурентное преимущество:**
- У конкурентов (Той Арт, Колокол) нет интерактивных сайтов
- Skeramos выделяется уникальным сочетанием студии и отеля
- Премиальный сайт позволяет продавать услуги дороже

**Визуальный стиль:**
- Стиль: Этно-Нео (современный + кыргызские традиционные мотивы)
- Логотип: спираль/нить — ключевой визуальный элемент
- Цветовая схема:
  - Творчество: темно-зеленый (#2D5016 → уточнить)
  - Отель: темно-красный/бордовый (#8B1538 → уточнить)
  - Акценты: золотой/охра (этно-элементы)
- Типографика: современная с этно-акцентами

**Контент:**
- Весь контент предоставляет клиент (фото, видео, тексты, цены)
- AI-элементы: уникальные анимированные паттерны через Gemini

## Constraints

- **Tech stack**: Будет определён после research-фазы (Next.js/Astro + headless CMS предпочтительно)
- **Hosting**: Нужен хостинг с поддержкой админ-панели
- **Content**: Клиент предоставляет весь медиа-контент
- **AI Assets**: Элементы дизайна генерируются через Gemini (первый/последний кадр → видео → фреймы)
- **Performance**: Сайт должен быстро грузиться несмотря на анимации
- **Mobile**: Mobile-first, критично для Бишкека (основной трафик с телефонов)
